# GAI-P3. 사람 얼굴 생성 고도화 (성능 향상)
## 프로젝트 개요
* 진행 일정 : **2024.07.20 (토) ~ 08.11 (일), 23일간**
* 프로젝트 목표 : 생성형 AI (GAN, CVAE, NVAE 등) 를 이용하여, 진짜 같은 사람 얼굴을 만드는 것

  * [NVAE official paper](https://arxiv.org/pdf/2007.03898)

  * 해당 모델을 이용하여 NVAE로 생성한 이미지를 리터칭
