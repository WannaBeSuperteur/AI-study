# GAI-P4. 폰트 생성 요청을 해석하여 폰트 생성
## 프로젝트 개요
* 진행 일정 : **2024.09.13 (금) ~ 09.22 (일), 10일간**
* 프로젝트 목표 :
  * 사용자의 폰트 생성 요청을 NLP 기술을 이용하여 해석, 중요한 내용을 파악
  * 해당 중요한 내용을 폰트 생성을 위한 Conditional VAE에 입력하여, 사용자의 요청에 맞는 폰트 (숫자 및 알파벳) 를 생성
