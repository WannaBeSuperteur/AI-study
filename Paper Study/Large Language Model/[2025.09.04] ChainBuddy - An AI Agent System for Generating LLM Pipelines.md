## 1. ChainBuddy 개요 및 핵심 아이디어

**ChainBuddy** 는 다음과 같은 **챗봇 형태의 Assistant 인터페이스** 이다.

| 구분 | 설명                                                                        |
|----|---------------------------------------------------------------------------|
| 목적 | LLM pipeline 생성 자동화                                                       |
| 특징 | - 유연하고 사용자 친화적<br>- 단순 prompt engineering 을 넘어선, **다양한 use case 를 커버 가능** |
