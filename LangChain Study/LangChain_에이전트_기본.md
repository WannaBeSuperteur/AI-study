## 1. LangChain 의 Agent 개요

**Agent (에이전트)** 는 거대 언어 모델 (LLM) 과 도구 (tools) 가 결합된 시스템이다.

* LLM 자체와 다른, Agent만의 특징은 다음과 같다.
  * 사용자 질문 분석을 통해, **어떤 도구를 사용할지 자율적으로 결정**
  * 도구 실행을 통해 얻은 결과를 바탕으로 **추가 작업 수행 여부 판단**
